# Installation Rapide - Chastity Tracker

## Étapes d'installation

### 1. Copier les fichiers
Copiez le dossier `chastity_tracker` dans :
```
votre-forum/ext/verturin/chastitytracker/
```

### 2. Activer l'extension
1. Connectez-vous au panneau d'administration (ACP)
2. **Personnalisation** → **Gestion des extensions**
3. Trouvez **Chastity Tracker**
4. Cliquez sur **Activer**

### 3. Configurer les permissions
1. **Permissions** → **Permissions des groupes**
2. Sélectionnez les groupes autorisés
3. Accordez les permissions :
   - ☑ Can view chastity tracker
   - ☑ Can manage own chastity periods
   - ☑ Can moderate chastity periods (pour les modérateurs)

### 4. Configurer les paramètres
1. **Extensions** → **Chastity Tracker** → **Paramètres**
2. Activez l'affichage sur les profils si souhaité

## Utilisation rapide

### Pour démarrer une période :
1. Panneau de l'utilisateur → **Chastity Tracker** → **Calendrier**
2. Sélectionnez la date de début
3. Cliquez sur **Soumettre**

### Pour terminer une période :
1. Calendrier → Bouton **Terminer la période**
2. Confirmez

### Pour voir les statistiques :
1. Panneau de l'utilisateur → **Chastity Tracker** → **Statistiques**
